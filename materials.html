Learning Materials

This page contains carefully prepared learning resources for the Python for Data Analysis program. Learners are encouraged to download the materials ahead of each session and follow along during practical demonstrations.

Session 1: Python Fundamentals for Data Analysis

The first session focuses on building a strong foundation in Python. You will learn how Python works, how to write and run scripts, and how to use core data structures commonly applied in data analysis tasks.

Before attending the session, please download the Session 1 learning guide (PDF). This document will be used throughout the class for live practice, explanations, and exercises.

To support hands-on learning, practice files have been prepared. Click to download the Python practice files and keep them on your computer. These files will help you practice writing scripts, working with lists and dictionaries, and running code in VS Code.

Additional reference notes are also available. You may download the Python reference notes to revise concepts after the session or when practicing independently.

New learning materials will be added after each session. Learners are advised to check this page regularly for updates.
